# Widget List #

Brief outline of widget types/components.

## Widget Types ##

-  Toggler
-  Select List
-  Select Buttons
-  Zoom Preview

## Widget Components ##

- Toggler
  +  button
  +  content (multiple ok)
- Select List
  +  select
  +  content
- Select Buttons
  +  select
  +  increment (button)
  +  decrement (button)
- Zoom Preview
  +  container
  +  normal-image
  +  zoom-image
  +  button
  +  thumbnails (optional)
