Carousels

This widget allows you to bind increment/decrement buttons to trigger the appropriate changes on a <select> element.

How To:

  1. Group the select-buttons components. (Add link here)
  2. Set the 'data-ur-select-buttons-component' attribute to 'select' on the target <select> tag.
  3. Set the 'data-ur-select-buttons-component' attribute to 'increment' on the increment button.
  4. Set the 'data-ur-select-buttons-component' attribute to 'decrement' on the decrement button.

Note : When there are no more options to increment/decrement the button will have the 'data-ur-state' attribute set to 'disabled'. Otherwise, it will have the 'enabled' value. It is recommended that the decrement button have this attribute set to 'disabled' in the page source.
