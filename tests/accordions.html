Accordions

How To:

Demo of the accordion widgets. The only thing required to setup accordions is to setup you're html properly. Then, onload, the accordion.js file combs the html for the elements you've declared as accordions and sets them up. To see how these accordions were setup, you should look at the page source, since the js added classes onload.

To setup an accordion:

  1. Add the mw_accordion attribute to the button element and all of the top level content elements. Set its value to be the name of the accordion (which should be unique to the page).
  2. Add the mw_button attribute to the button element (no value required).
  3. Add the mw_content attribute to the content element(s) (no value required).
