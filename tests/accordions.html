Accordions

How To:

Demo of the accordion widgets. The only thing required to setup accordions is to setup you're html properly. Then, onload, the accordion.js file adds the listeners appropriately.

To setup an accordion:

  1. Add the 'mw_accordion' attribute to the button element and all of the top level content elements. Set its value to be the name of the accordion (which should be unique to the page).
  2. Add the 'mw_accordion_button' (and optionally 'closed') classes to the button element.
  3. Add the 'mw_accordion_content' (and optionally 'closed') classes to the content element(s).
