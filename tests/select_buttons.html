Select Buttons

This widget allows you to bind increment/decrement buttons to trigger the appropriate changes on a <select> element.

How To:

  1. Set the 'select-buttons' attribute to a unique name. This should be set on the target <select> tag as well as the buttons.
  2. Add the 'mw_button' and either 'mw_increment' or 'mw_decrement' classes to the buttons as appropriate.

Note : When there are no more options to increment/decrement the button will have the 'disabled' class added to it.
