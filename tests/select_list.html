Select List

This widget allows you to bind a non-select list to trigger changes on a <select> element.

How To:

  1. Set the 'select-list' attribute to a unique name. This should be set on the target <select> tag as well as the trigger list element.
  2. Make sure that the elements in your target list have the same value attributes as the original <option> elements from the target <select>

Tip: In v2, the easiest way to construct a trigger list from a <select> element would be to copy the <select> element, then change its tag name, and its children's tag names.
