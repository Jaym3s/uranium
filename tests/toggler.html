Togglers

Description:

Toggler is a widget that has two components -- a button, and a set of contents. When you click the button, the button and content states are toggled.

How To:

Demo of the toggler widgets. The only thing required to setup togglers is to setup you're html properly. Then, onload, the toggler.js file adds the listeners appropriately.

To setup an toggler:

  1. Group the togglers:
    • Group by Structure:

      To group by structure, set the 'data-ur-set' attribute to 'toggler' on an element that contains the button and all the content elements.

    • Group by Unique ID:

      To group by unique ID, set the 'data-ur-id' to a value unique to the components you'd like to group. It can have any value, the only condition is that its unique to the group.

  2. Set the 'data-ur-toggler-component' attribute to 'button' on the button element. Optionally -- add the 'data-ur-state' attribute (to 'disabled' or 'enabled' ). By default it will be set to 'enabled' onload.
  3. Set the 'data-ur-toggler-component' attribute to 'content' on the content element(s). Optionally -- add the 'data-ur-state' attribute (to 'disabled' or 'enabled' ). By default it will be set to 'enabled' onload. If you so desire, you can add different states to different content elements. Toggler always toggles the states, no matter what they are.
