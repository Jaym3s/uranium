Togglers

How To:

Demo of the toggler widgets. The only thing required to setup togglers is to setup you're html properly. Then, onload, the toggler.js file adds the listeners appropriately.

To setup an toggler:

  1. Add the 'mw_toggler' attribute to the button element and all of the top level content elements. Set its value to be the name of the toggler (which should be unique to the page).
  2. Add the 'mw_toggler_button' (and optionally 'closed') classes to the button element.
  3. Add the 'mw_toggler_content' (and optionally 'closed') classes to the content element(s).
